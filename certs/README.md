# Сертификаты и ключи

Эта директория предназначена для хранения:

## Для iOS (APNS)
- `apns-key.p8` - Ключ Apple Push Notification Service

Как получить:
1. Войдите в Apple Developer Account
2. Перейдите в Certificates, Identifiers & Profiles
3. Создайте новый ключ (Keys) с возможностью APNs
4. Скачайте .p8 файл и поместите его сюда

## Для Android
Android использует собственное WebSocket соединение, поэтому никаких дополнительных сертификатов не требуется.

**Внимание!** Эти файлы не должны попадать в git репозиторий.
