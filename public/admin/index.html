SDK & API

Web Push

Интеграция для браузеров

<script src="/sdk/push-sdk.js"></script> <script> PushSDK.init({ apiKey: 'YOUR_API_KEY', vapidPublicKey: 'YOUR_VAPID_KEY' }); PushSDK.subscribe({ userId: 'user123', tags: ['news'] }); </script>

iOS (Swift)

Apple Push Notification Service

// AppDelegate.swift func application(_ app: UIApplication, didRegisterForRemoteNotificationsWithDeviceToken deviceToken: Data) { let token = deviceToken.map { String(format: "%02.2hhx", $0) }.joined() registerDevice(token: token) }

Android (WebSocket)

Собственная реализация без Google

// Application.kt import com.yourcompany.pushsdk.PushSDK class MyApp : Application() { override fun onCreate() { super.onCreate() PushSDK.init( this, "wss://your-server.com/ws/android", "YOUR_API_KEY" ) PushSDK.getInstance().connect() } }
